Module: controllers/empleadoController

Controller de Empleados. Gestiona las operaciones CRUD para empleados, operando sobre un modelo de tabla dividida (Usuario para autenticación y Empleado para datos legajales y laborales). Implementa control de acceso estricto por Espacio de Trabajo.
