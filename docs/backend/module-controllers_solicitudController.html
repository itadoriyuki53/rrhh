Module: controllers/solicitudController

Controller de Solicitudes. Maneja las operaciones CRUD para solicitudes de recursos humanos, incluyendo vacaciones, licencias, horas extras y renuncias. Utiliza servicios de validación específicos por tipo y helpers de permisos y workspace para la separación de responsabilidades.
